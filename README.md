# AmygNet

A segmentation model for segmenting extremely small brain structures, e.g., amygdala and its subnuclei.

## Getting Started

### Prerequisites
'''
python 2.7
PyTorch 0.4.1
Cuda 10.0+
'''


### Installing
(for Waisman Users)
'''
pip install torch==0.4.1 -f https://download.pytorch.org/whl/cu100/stable --user
'''
